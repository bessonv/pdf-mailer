# PDF Mailer web app

Simple app that sends pdf document to the list of recipients. Can send to mail or telegram_id.
To receive telegram messages recipients must start a chat with @pdfmailer_bot.

## Uses
- PHP API server
- Sqlite database
- React client

## Composer packages:
- phpmailer/phpmailer
- longman/telegram-bot

## Npm packages:
- react
- react-router-dom
- vite
